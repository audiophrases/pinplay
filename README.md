# Quiz Creator Guide

## Comprehensive Overview
This guide provides an in-depth explanation of the quiz creator tool, detailing functionalities, features, and limitations.

## Question Type Specifications
- **Multiple Choice**: Allows users to select one correct answer from a list.
- **True/False**: Simple binary option.
- **Short Answer**: Text input with no set limit.
- **Essay**: Extended text input for detailed responses.

## TTS (Text-to-Speech) Features
- Supports multiple languages and voices.
- Customizable speed and pitch.
- Preview option before finalizing.

## Auto-Image Generation
- Automatically generates relevant images based on questions.
- Utilizes AI algorithms to create context-appropriate visuals.

## Maximum Limitations
- Max questions per quiz: 50.
- Max characters per question: 200.
- Max TTS audio length: 5 minutes.
- Image size limit: 5MB.

## Builder Features
- Drag-and-drop interface for easy question arrangement.
- Preview mode to test quizzes before publication.
- Export options for sharing or printing quizzes.

## Conclusion
This guide aims to equip you with the knowledge to effectively use the quiz creator tool, ensuring a seamless experience in creating engaging quizzes.
