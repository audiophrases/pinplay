# PinPlay Teacher Guide (Batch-by-Batch)

Goal: build a practical, creative manual teachers can use immediately, with ideas that make learning fun and effective.

---

## Batch 1 — Multiple Choice (MCQ)

How to read this section:
- Each case includes: **Learning goal**, **How to run it**, and **Creative twist**.
- All cases are designed so teachers can copy/adapt in minutes.

### 1) Misconception Hunt
- **Learning goal:** Diagnose common misunderstandings early.
- **How to run it:** Build 8–10 MCQs where wrong options are realistic misconceptions.
- **Creative twist:** After each reveal, ask students to defend why the wrong option looked tempting.

### 2) Two-Stage Confidence Quiz
- **Learning goal:** Improve metacognition and self-awareness.
- **How to run it:** First question: pick answer. Second question: “How sure are you?” (MCQ confidence scale).
- **Creative twist:** Celebrate “low confidence but correct” to reward honest reflection.

### 3) Storyline Mission Quiz
- **Learning goal:** Increase engagement in content review.
- **How to run it:** Wrap 10 MCQs inside a story (escape room, detective case, space mission).
- **Creative twist:** Name sections as “Level 1/2/3” and add dramatic prompts between questions.

### 4) Near-Miss Vocabulary Training
- **Learning goal:** Build precision with similar words/forms.
- **How to run it:** Use near-synonyms/false friends as distractors (especially useful for ESL).
- **Creative twist:** Ask students to create one extra distractor after reveal.

### 5) Data-to-Decision MCQ
- **Learning goal:** Apply interpretation skills, not memorization.
- **How to run it:** Show mini data in the prompt (short table/text/stat), then ask for best conclusion.
- **Creative twist:** Include one “technically true but irrelevant” distractor.

### 6) Exam Strategy Sprint
- **Learning goal:** Train test-taking strategy under time pressure.
- **How to run it:** 10 quick MCQs with short timer and mixed difficulty.
- **Creative twist:** Pause after every 3 questions for a 30-second strategy debrief.

### 7) Error-Analysis Round
- **Learning goal:** Learn from wrong answers systematically.
- **How to run it:** After reveal, students categorize errors: vocabulary / grammar / logic / rush.
- **Creative twist:** Keep a class “error heatmap” and target next lesson by top error category.

### 8) Reverse MCQ (Student as Teacher)
- **Learning goal:** Deepen understanding by question design.
- **How to run it:** Students write one MCQ each; teacher imports best ones next class.
- **Creative twist:** Award bonus points for best distractor quality, not only correct answer.

### 9) Real-Life Scenario Choice
- **Learning goal:** Transfer learning to authentic contexts.
- **How to run it:** Prompt with realistic situations (email reply, travel issue, science claim, civic dilemma).
- **Creative twist:** Use “best option” instead of “only correct option” to spark richer discussion.

### 10) Retrieval Spiral (Spacing)
- **Learning goal:** Improve long-term retention.
- **How to run it:** Mix today’s topic with 20–30% old-topic MCQs from previous weeks.
- **Creative twist:** Tag questions as “Today” vs “Throwback” to normalize revision.

---

## Reusable MCQ Design Rules (Teacher Cheat Sheet)

1. Keep stems clear and short (one idea per question).
2. Make distractors plausible (avoid obvious nonsense).
3. Avoid grammar clues that reveal correct answer.
4. Balance option length (don’t make correct answer always longest).
5. Use reveals for mini-teaching, not just scoring.
6. Mix cognitive levels: recall, apply, analyze.
7. Use quick timers for energy, longer timers for reasoning-heavy items.
8. Recycle strong distractors in future quizzes to test true learning.
9. Track frequent wrong options to guide remediation.
10. End with 1 reflective prompt: “What fooled you today?”
